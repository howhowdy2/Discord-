# Discord 定時指令

## 安裝方式
1. 安裝 [Tampermonkey](https://www.tampermonkey.net/)

2. 點擊 `discord_scheduler_v3.user.js` → Raw → Tampermonkey 會自動偵測並詢問安裝  
   或者 直接將`discord_scheduler_v3.user.js`內容貼至`Tampermonkey`內


## 功能
- 右上角新增「定時指令」按鈕
- 支援 Bot 指令 / 一般對話兩種模式
- 可同時設定多個獨立計時指令
